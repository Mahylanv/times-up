# Times Up Sprint

Jeu Times Up rapide en 3 rounds avec cartes aleatoires, equipes variables et mode PWA.

## Lancer en local

```bash
npm install
npm run dev
```

## Build & run production

```bash
npm run build
npm run start
```

## Docker

```bash
docker build -t timesup .
docker run -p 3000:3000 timesup
```

## Docker Compose

```bash
docker compose up --build
```
